WIN — Student Goal Tracker and Reporting

Track goals, document services, and log critical incidents with auditable workflows designed for student support programs.

Project Overview

WIN Student Goal Tracker supports student goal tracking, services documentation, and critical incident reporting with auditability and privacy in mind. The system aims to reduce manual paperwork, standardize documentation, and improve visibility for program staff.

Goal Tracking Create, track, and update student goals over time.
Documentation Case notes and services documented consistently.
Incident Logging Record incidents with structured fields and history.

Core Modules

Student goals, case notes, services provided, and critical incident reports—designed around day-to-day workflows for advisors, mentors, and case managers.
